## Geral
```
Cloud Computing: Ganhar tempo
Conseito, compartilhamento de recursos configurávceis, 
esforço mínimo para gerenciamento, não necessita a interação do provedor.

Crescimento Vertical   - Servidor comum, PC, aumenta a memória
Crescimento Horizontal - Ele criar mais servidores de acordo com a necessidade

***********************************************************
IaaS - Infraestrutura como serviço. Uma rede virtualizada, utiliza o que precisa e paga por isso.
AWS, Azure, OpenStack, IBM SoftLayer, Google.

PasS - Plataforma como serviço
AWS RDS, Google App Engine
AWS RDS - Banco de dados pronto como serviço, ele sobe o servidor MySQL e etc.

SaaS - Software como serviço - IaaS e PasS para criar o SaaS
Gmail, Skype, Trello, Slack
***********************************************************

Modelos de deploy:
Nuvens privadas     - só a sua empresa utiliza
Nuvens pública      - a grande maioria, um provedor vende recursos de nuvem para qualquer pessoa utilizar, recurso de provedores publicos
Nuvens comunitária  - privada que mais empresas podem compartilhar  
Nuvens híbrida      - privada/publica
```
